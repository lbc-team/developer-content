按 lamport 余额返回 20 个最大的帐户（结果最多可以缓存两个小时）

<DocSideBySide>

<DocLeftSide>

### 参数

<Parameter type={"object"} optional={true}>

配置对象包含以下字段：

<Field
  name="commitment"
  type="string"
  optional={true}
  href="/docs/rpc/index.zh.mdx#配置状态承诺"
></Field>

<Field name="filter" type="string" optional={true}>
  按帐户类型过滤结果
  <Values values={["circulating", "nonCirculating"]} />
</Field>

</Parameter>

### 结果

结果将是一个 RpcResponse JSON 对象，其`value`等于包含以下内容的数组 `<object>`：

- `address: <string>` - 账户的base-58编码地址
- `lamports: <u64>` - 帐户中的lamport数量，作为u64

</DocLeftSide>

<DocRightSide>

### 代码示例

```bash
curl http://localhost:8899 -X POST -H "Content-Type: application/json" -d '
  {"jsonrpc":"2.0","id":1, "method":"getLargestAccounts"}
'
```
