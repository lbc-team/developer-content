返回给定槽位范围的槽位前导者

<DocSideBySide>

<DocLeftSide>

### 参数

<Parameter type={"u64"} optional={true}>
  起始槽，作为 u64 整数
</Parameter>

<Parameter type={"u64"} optional={true}>
  限制，作为 u64 整数（1 到 5,000 之间）
</Parameter>

### 结果

`<array[string]>` - 作为 base-58 编码字符串的节点身份公钥数组

</DocLeftSide>

<DocRightSide>

### 代码示例

如果当前位置是`#99` - `10`使用以下请求查询下一个领导者：

```bash
curl http://localhost:8899 -X POST -H "Content-Type: application/json" -d '
  {
    "jsonrpc":"2.0", "id": 1,
    "method": "getSlotLeaders",
    "params": [100, 10]
  }
'
```
