返回一个 epoch 的领导者时间表

<DocSideBySide>

<DocLeftSide>

### 参数

<Parameter type={"u64"} optional={true}>

  获取与所提供的 slot 相对应的 epoch 的领导者时间表。

<Callout type={"info"}>如果未指定，则获取当前时期的领导者计划</Callout>

</Parameter>

<Parameter type={"object"} optional={true}>

配置对象包含以下字段：

<Field
  name="commitment"
  type="string"
  optional={true}
  href="/docs/rpc/index.zh.mdx#配置状态承诺"
></Field>

<Field name="identity" type="string" optional={true}>
  仅返回此验证者身份的结果（base-58 编码）
</Field>

</Parameter>

### 结果

返回具有以下两个值之一的结果：

- `<null>` - 如果未找到请求的 epoch，或者
- `<object>` - 结果字段将是验证者身份的字典，作为 base-58 编码字符串，及其相应的
  领导者 slot 索引作为值（索引相对于所请求的 epoch 中的第一个 slot）

</DocLeftSide>

<DocRightSide>

### 代码示例

```bash
curl http://localhost:8899 -X POST -H "Content-Type: application/json" -d '
  {
    "jsonrpc": "2.0",
    "id": 1,
    "method": "getLeaderSchedule",
    "params": [
      null,
      {
        "identity": "4Qkev8aNZcqFNSRhQzwyLMFSsi94jHqE8WNVTJzTP99F"
      }
    ]
  }
'
```
