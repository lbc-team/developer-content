返回从给定 slot 开始的已确认块的列表

<DocSideBySide>

<DocLeftSide>

### 参数

<Parameter type={"u64"} required={true}>
  start_slot，`u64`整数
</Parameter>

<Parameter type={"u64"} required={true}>
  限制，作为`u64`整数（必须比 更高不超过 500,000 个块 `start_slot`）
</Parameter>

<Parameter type={"object"} optional={true}>

配置对象包含以下字段：

  <Field
  name="commitment"
  type="string"
  optional={true}
  defaultValue="finalized（已完成）"
  href="/docs/rpc/index.zh.mdx#配置状态承诺"
>

- 不支持"processed"

</Field>

</Parameter>

### 结果

结果字段将是一个 u64 整数数组，列出已确认的块，从 开始`start_slot`最多`limit`可
达块（含）。

</DocLeftSide>

<DocRightSide>

### 代码示例

```bash
curl http://localhost:8899 -X POST -H "Content-Type: application/json" -d '
  {
    "jsonrpc": "2.0",
    "id":1,
    "method":"getBlocksWithLimit",
    "params":[5, 3]
  }
'
```

### 响应

```json
{
  "jsonrpc": "2.0",
  "result": [5, 6, 7],
  "id": 1
}
```

</DocRightSide>
