返回节点在其账本中拥有相关信息的最低槽位。

<Callout type={"info"}>
  如果节点配置为清除旧账本数据，则该值可能会随着时间的推移而增加
</Callout>

<DocSideBySide>

<DocLeftSide>

### 参数

**没有任何**

### 结果

`u64` - 最小账本槽数

</DocLeftSide>

<DocRightSide>

### 代码示例

```bash
curl http://localhost:8899 -X POST -H "Content-Type: application/json" -d '
  {"jsonrpc":"2.0","id":1, "method":"minimumLedgerSlot"}
'
```

### 响应

```json
{ "jsonrpc": "2.0", "result": 1234, "id": 1 }
```

</DocRightSide>
